# Open Source Positioning

## Public Promise

Atlas is MIT-licensed and open to contributors for self-hosting, integrations, and UI/product improvements.

## What Stays Proprietary

- Brand and trademarks
- Hosted SaaS operational tooling
- Optional enterprise/private deployments

## Messaging

- "Open core, practical defaults, production path included."
- "Self-host first, SaaS optional."
- "MIT licensed to reduce legal friction for teams."

## Contribution Framing

- Accept issues and PRs for docs, integrations, UX, and deployment improvements.
- Keep architecture docs and roadmap public.
- Use small, atomic PRs and clear changelog notes.
